## 🚀 Установка и запуск на Windows

### Предварительные требования

- Python 3.10 или выше
- pip (менеджер пакетов Python)

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/replicaCortex/signalis

```

```bash
pip install numpy raylib
```

python src/main.py

### Альтернативный способ (без виртуального окружения)

```bash
pip install numpy raylib

```

## 📐 Используемые математические формулы

### 1. Статистические характеристики (`analysis/stats.py`)

**Полная мощность сигнала:**
$$P_{total} = \sum_{i=1}^{n} x_i^2$$

**Средняя мощность:**
$$\bar{P} = \frac{1}{n}\sum_{i=1}^{n} x_i^2$$

**Математическое ожидание (среднее):**
$$\mu = \frac{1}{n}\sum_{i=1}^{n} x_i$$

**Дисперсия:**
$$\sigma^2 = \frac{1}{n}\sum_{i=1}^{n} (x_i - \mu)^2$$

**Среднеквадратическое отклонение (СКО):**
$$\sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n} (x_i - \mu)^2}$$

### 2. Спектральный анализ (`analysis/spectral.py`)

**Автокорреляционная функция (ACF):**
$$R_{xx}[k] = \frac{\sum_{i=0}^{n-k-1} (x_{i+k} - \mu)(x_i - \mu)}{R_{xx}[0]}$$
где $R_{xx}[0]$ - значение автокорреляции при нулевом сдвиге (дисперсия)

**Спектральная плотность мощности (PSD) через БПФ:**
$$P(f) = \frac{|X(f)|^2}{N}$$
где $X(f)$ - преобразование Фурье сигнала, $N$ - количество точек

### 3. Генерируемые сигналы (`signals/deterministic.py`)

**Гауссов импульс:**
$$g(t) = A \cdot \exp\left(-\frac{(t - t_0)^2}{2\sigma^2}\right)$$

**Пилообразный сигнал:**
$$s(t) = A \cdot (2(f \cdot t - \lfloor f \cdot t + 0.5 \rfloor))$$

**Полигармонический сигнал:**
$$p(t) = \sum_{k} A_k \sin(2\pi f_k t)$$

### 4. Шум и SNR (`signals/random.py`)

**Отношение сигнал/шум (SNR) в дБ:**
$$SNR_{dB} = 10 \log_{10}\left(\frac{P_{signal}}{P_{noise}}\right)$$

**Мощность шума для заданного SNR:**
$$P_{noise} = \frac{P_{signal}}{10^{SNR_{dB}/10}}$$

**Белый гауссов шум:**
$$n(t) \sim \mathcal{N}(0, \sqrt{P_{noise}})$$

## 🎮 Управление

Интерфейс содержит следующие кнопки:

- **Gaussian** - генерация гауссова импульса
- **Sawtooth** - генерация пилообразного сигнала
- **Polyharmonic** - генерация полигармонического сигнала
- **+Noise** - добавление шума к текущему сигналу
- **Load WAV** - загрузка сигнала из WAV файла (input.wav)
- **Save WAV** - сохранение сигнала в WAV файл (output.wav)
- **Analyze** - вычисление статистик и спектров

## 📈 Отображение данных

- **Signal** - график исходного сигнала (синий)
- **ACF** - автокорреляционная функция (зеленый)
- **PSD** - спектральная плотность мощности (красный)
- **Statistics** - таблица со статистическими характеристиками
