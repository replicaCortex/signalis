## 🚀 Установка и запуск на Windows

### Предварительные требования

- Python 3.10 или выше
- [uv](https://docs.astral.sh/uv/) (менеджер пакетов)

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/replicaCortex/signanlis
cd signalis
```

### Шаг 2: Установка зависимостей

```bash
uv sync
```

### Шаг 3: Запуск приложения

```bash
uv run signalis
```

### Альтернативный способ (без uv)

```bash
python -m venv venv
venv\Scripts\activate
pip install PyQt5 matplotlib numpy
python -m signalis
```

## 📐 Используемые математические формулы

### 1. Генерация сигналов (`dsp.py`)

**Гармонический сигнал (сумма гармоник):**
$$x(t) = \sum_{k} A_k \cos(2\pi f_k t + \varphi_k)$$

**Гауссов импульс:**
$$g(t) = A \cdot \exp\left(-\frac{(t - t_0)^2}{2\sigma^2}\right)$$

где $t_0 = T/2$ — центр импульса (середина записи), $\sigma = \frac{1}{2\pi f}$ — ширина импульса, определяемая частотой.

**Пилообразный сигнал:**
$$x(t) = A \cdot \left(2 \cdot \frac{t \bmod T_p}{T_p} - 1\right)$$

Линейно нарастает от $-A$ до $+A$ за один период $T_p = 1/f$.

**Случайная помеха:**
$$w(t) = A_w \cdot \cos(2\pi f_w t), \quad A_w \sim U[A_{min},\; A_{max})$$

где $U$ — равномерное распределение целых чисел.

### 2. Дискретное преобразование Фурье (`dsp.py`)

**Прямое ДПФ:**
$$X_{Re}[k] = \sum_{n=0}^{N-1} x[n] \cos\left(\frac{2\pi k n}{N}\right)$$

$$X_{Im}[k] = \sum_{n=0}^{N-1} x[n] \sin\left(\frac{2\pi k n}{N}\right)$$

**Нормировка:**
$$X_{Re}[0] = \frac{X_{Re}[0]}{N}, \quad X_{Re}[k] = \frac{2 \cdot X_{Re}[k]}{N} \quad (k \geq 1)$$

**Амплитудный спектр:**
$$|X[k]| = \sqrt{X_{Re}^2[k] + X_{Im}^2[k]}$$

**Спектральная плотность мощности (при наличии помехи):**
$$S[k] = \frac{|X[k]|^2}{N}$$

**Фазовый спектр:**
$$\varphi[k] = -\arctan\left(\frac{X_{Im}[k]}{X_{Re}[k]}\right)$$

### 3. Цифровые фильтры (`filters.py`)

**Общая рекурсивная формула IIR-фильтра:**
$$y[n] = a_0 x[n] + a_1 x[n-1] + a_2 x[n-2] - b_1 y[n-1] - b_2 y[n-2]$$

**Коэффициенты фильтров:**

| Фильтр | $a_0$ | $a_1$ | $a_2$ | $b_1$ | $b_2$ |
|--------|-------|-------|-------|-------|-------|
| Хеннинга | $\frac{1}{4}$ | $\frac{1}{2}$ | $\frac{1}{4}$ | $0$ | $0$ |
| Первого порядка | $1$ | $0$ | $0$ | $-r$ | $0$ |
| НЧФ | $1$ | $2$ | $1$ | $-2r\cos\omega_c$ | $r^2$ |
| ВЧФ | $1$ | $-2$ | $1$ | $-2r\cos\omega_c$ | $r^2$ |
| Полосовой | $1$ | $0$ | $-1$ | $-2r\cos\omega_c$ | $r^2$ |
| Режекторный | $1$ | $-2\cos\omega_c$ | $1$ | $-r\cos\omega_c$ | $r^2$ |

где $\omega_c = 2\pi f_c / f_d$, $r$ — коэффициент затухания.

**Параболический фильтр (5-точечное сглаживание):**
$$y[n] = \frac{-3x[n-2] + 12x[n-1] + 17x[n] + 12x[n+1] - 3x[n+2]}{35}$$

### 4. Амплитудно-частотная характеристика (`filters.py`)

$$H(f) = \sqrt{\frac{\left(a_0 + a_1\cos\omega + a_2\cos 2\omega\right)^2 + \left(a_1\sin\omega + a_2\sin 2\omega\right)^2}{\left(1 + b_1\cos\omega + b_2\cos 2\omega\right)^2 + \left(b_1\sin\omega + b_2\sin 2\omega\right)^2}}$$

где $\omega = 2\pi j / N$.

## 🎮 Управление

### Панель параметров

- **Время T** — длительность сигнала в секундах
- **Частота дискретизации Fd** — определяет шаг $\Delta T = 1/F_d$ и количество точек $N = T / \Delta T$
- **Тип сигнала** — выбор формы: гармонический, гауссов импульс, пилообразный
- **Таблица компонент** — частота, амплитуда и фаза каждой компоненты (фаза используется только для гармонического сигнала)

### Кнопки

- **Моделировать** — генерация сигнала и вычисление спектра
- **Очистить таблицу** — сброс таблицы компонент
- **Добавить помеху** — наложение случайного шума на сигнал
- **Применить фильтр** — цифровая фильтрация с выбранным типом фильтра

## 📈 Отображение данных

### Вкладка «Сигналы»

Временные графики с чекбоксами:

- **Базовый сигнал** — чистый гармонический / гауссов импульс / пилообразный 
- **Помеха** — случайный шум 
- **Результирующий** — сумма базового и помехи 
- **Отфильтрованный** — сигнал после фильтрации 

### Вкладка «Амплитудный спектр»

Столбчатая диаграмма $|X[k]|$ или $S[k]$ по частоте. При включённом фильтре — спектр отфильтрованного сигнала накладывается поверх.

### Вкладка «Фазовый спектр»

Столбчатая диаграмма $\varphi[k]$ в градусах по частоте.

### Вкладка «АЧХ фильтра»

Коэффициент передачи $H(f)$ выбранного фильтра. Отображается после нажатия «Применить фильтр».
